Shreya L Naik — Portfolio (HTML/CSS/JS)
========================================

This is a standalone portfolio built with plain HTML, CSS, and JavaScript.
No build tools, no Vite, no installation required.

HOW TO USE
----------
1. Unzip the folder.
2. Double-click "index.html" — it will open in your browser. That's it!

FILES
-----
- index.html        → main page structure
- styles.css        → all styling (dark navy hero + white sections)
- script.js         → interactivity (menu, scroll effects, content data)
- assets/           → your photo + resume PDF

EDITING CONTENT
---------------
Open script.js to update:
- Skills (technicalSkills array)
- Tools (tools array)
- Projects (projects array)
- Education (timeline array)
- Certifications (certifications array)

Open index.html to update contact email, phone, and location
(search for "shreyalnaik@example.com").

HOSTING (free options)
----------------------
- GitHub Pages: push the folder to a repo, enable Pages in settings.
- Netlify Drop: drag & drop the folder at https://app.netlify.com/drop
- Vercel: import the folder.
